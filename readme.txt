WP Club Manager for Cricket is an extension for WP Club Manager which adds extra features for cricket clubs managing their website with WordPress.

== Description ==
= Build A Pro Cricket Club Website =
WP Club Manager for Cricket is an extension for [WP Club Manager](http://wordpress.org/plugins/wp-club-manager/) that includes features specific to cricket including better match scorecards, player styles and improved overs handling to help your cricket club build an awesome website.

= Features Include =

* Detailed cricket style scorecards
* Player batting and bowling styles
* Batting and bowling stats
* Strike rate and bowling economies
* Record extras and overs
* Match toss
* Works seamlessly with WP Club Manager
* Customize appearance with flexible templates

== Installation ==

= Minimum Requirements =

* WordPress 4.2 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WPCM for Cricket, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “WP Club Manager for Cricket” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work as expected but as always, it is recommended that you backup your site just in case any problems should occur during the update process.

== Frequently Asked Questions ==
= Will this plugin work on it's own? =

No. You must first install and activate the free [WP Club Manager](http://wordpress.org/plugins/wp-club-manager/) plugin, version 1.5.7 or above.
